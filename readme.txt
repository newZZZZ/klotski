Klotski Puzzle and Klotski Solver 
Play Klotski with three different openings 
Can ask computer to solve the Klotski at anytime 
Written in Ocaml, using the built-in module Graphics for graphical output and UI, 
and **imagelib** for loading PNG file. 
Need to install imagelib package
Use mouse to play the game

To Installation imagelib package:
Opam Install imagelib 

To build:
make
